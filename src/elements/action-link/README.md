## Usage

```html
<action-link
  target="_blank"
  href="/more-details"
>
  Find out more
</action-link>

<action-link
  class="custom-class"
  href="/more-details"
>
  Find out more
</action-link>
```

## Bindable properties

| Property | Type | Default |
|----------|------|---------|
|`class`|`string`|none|
|`href`|`string`|none|
|`target`|`string`|none|
