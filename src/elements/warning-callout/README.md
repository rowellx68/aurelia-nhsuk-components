## Usage

```html
<warning-callout
  class="some-other-class"
  label-prefix="Important:"
  label="School, nursery or work"
>
  <p>Stay away from school, nursery or work.</p>
</warning-callout>
```

## Bindable properties

| Property | Type | Default |
|----------|------|---------|
|`class`|`string`|none|
|`label-prefix`|`string`|none|
|`label`|`string`|none|
