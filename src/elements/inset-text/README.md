## Usage

```html
<inset-text class="some-other-class" text-prefix="Important:">
  <p>
    You can report any suspected side effect to the UK safety scheme.
  </p>
</inset-text>
```

## Bindable properties

| Property | Type | Default |
|----------|------|---------|
|`class`|`string`|none|
|`text-prefix`|`string`|none|
